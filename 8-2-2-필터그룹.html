8-2-2. 필터그룹

8. 설정 ›› 8-2. 정보 설정 ››

부서 및 네트워크 그룹은 고정된 정보값을 기준으로 그룹 구성을 합니다. 필터그룹은 DB쿼리를 이용한 조건에 의해 다이나믹한 그룹을 구성할 수 있습니다.

  • 설정방법은 [쿼리 관리자]와 유사한 구조로 되어 있습니다.

  • 단, 변수 설정이 없으며, SELECT 기준 컬럼값은 "ID" 컬럼을 기본으로 합니다.

  • ID 컬럼은 장비 정보를 저장하고 있는 DB 테이블의 장비 고유 정보를 갖고 있는 컬럼입니다.

  • 설정의 상세한 내용은 [쿼리 관리자]를 참고합니다.

설정방법 

  1. 그룹 추가 버튼을 클릭합니다.

  2. 선택한 그룹 하위 그룹으로 추가되므로 설정 시 주의합니다.

  3. 생성된 그룹을 선택하고 필터 추가 버튼을 클릭합니다.

  4. 추가된 필터를 선택하고 우측에 이름과 쿼리를 입력합니다.

  5. 쿼리 관리자와는 달리 별도 저장 버튼 없이 정보가 저장됩니다.

  6. 입력이 완료되면 실행 버튼을 클릭하여 결과를 확인합니다.

참고사항 

  1. 본 메뉴에서는 프린트 기능이 제공되지 않습니다.

  2. 필터 그룹 설정은 에이전트 정보의 특정값을 필터하여 자동 분류하기 위한 설정입니다. 

설정된 필터 그룹은 조직도의 필터 그룹 목록에서 나타납니다.
