8-2-6-2. 명령서버

8. 설정 ›› 8-2. 정보 설정 ›› 8-2-6. 서버관리 ››

명령서버는 에이전트들과 통신을 담당하며 명령과 처리 결과를 데이터 서버로 전달하는 역할을 합니다. 

하나 이상의 서버 구성이 가능하기 때문에 명령서버 메뉴는 이러한 명령서버들을 그룹단위로 관리할 수 있으며, 각 서버 그룹이 담당하고 있는 에이전트 IP대역을 설정하여 분배관리할 수 있습니다.

설정방법

  1. 명령서버 Tab을 선택한 뒤 추가버튼을 선택하면 새 서버그룹이 생성됩니다. 

  2. 서버목록 Tab을 선택한 뒤 추가 버튼을 클릭합니다.

  3. 서버그룹 : 생성한 서버 그룹을 선택합니다.

  4. 이름 : 서버 명칭을 입력합니다.
  5. 서버 IPv4 주소 : 서버 IP를 입력합니다.

  6. 최대 에이전트 개수 : 서버 접속 가능한 에이전트 수량을 입력합니다.

  7. 전체 서버에서 각 서버의 최대 에이전트 접속 수량이 초과할 경우 자동으로 분산 접속시킵니다.

  8. 우선순위 : 에이전트 접속 우선 수위를 설정합니다.

  9. 입력이 완료되면 확인 버튼을 클릭합니다.

  10. 추가된 내용을 확인 합니다.
  11. 담당 IP 대역 Tab을 선택한 뒤 추가버튼을 클릭합니다.

  12. IP대역 정보 등록/수정 화면이 팝업되면 해당 정보를 입력합니다. 

  13. 입력이 완료되면 확인 버튼을 클릭합니다.

  14. 추가된 정보를 확인합니다.
