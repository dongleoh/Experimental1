3-1-3-3. 공지사항

3. 공통 ›› 3-1. 도구 ›› 3-1-3. 알림센터 ››

에이전트에 표시되는 공지사항 정보를 등록 관리할 수 있습니다. 

공지사항 기능

  1. 추가 : 새로운 공지사항을 추가한다.
  2. 수정 : 공지사항을 수정한다.
  3. 삭제 : 공지사항을 삭제한다.

에이전트 공지사항 확인

[기본정보] 메뉴를 클릭한 뒤 공지사항 내용을 확인하고 조회 버튼을 클릭하면 공지내용을 확인할 수 있습니다.
