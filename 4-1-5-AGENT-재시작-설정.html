4-1-5. AGENT 재시작 설정

4. 일반정보 ›› 4-1. 장비관리 ››

에이전트는 매일 밤 12시에 재기동(시작)을 하도록 되어있습니다. 이 시간을 변경하고 싶은 경우 아래 기능을 사용합니다. 

아래 설정을 하게되면 밤12시 재기동은 동작하지 않습니다. 

설정방법

  1. 그룹을 추가합니다.

  3. 그룹 명과 에이전트를 재시작 할 시작시간, 종료시간을 설정합니다. (분 단위로 설정 가능)

  4. 그룹에 포함시킬 장비를 선택합니다.

  6. Agent 재시작 명령을 내립니다.

  7. 실제로 Agent 자동 재시작된 시간을 확인할 수있습니다. ("Agent 재시작 명령"을 통해 재시작된 경우는 표시되지 않습니다.)

참고사항

변경한 내용은 Agent 재시작 이후에 반영됩니다.
