5-1-1. 라이선스

5. 소프트웨어 ›› 5-1. 자산관리 ››

관리 소프트웨어 목록으로 제조사 및 상용유형으로 분류되어 있으며, 라이선스 정보를 관리할 수 있습니다.

또한, Excel 파일을 SWeeper DB로 업로드가 가능하므로, 기존의 구매자료를 쉽게 등록할 수 있습니다. 

소프트웨어라이선스 탭에서 검색 버튼을 클릭 하여 기존에 추가한 소프트웨어 라이선스를 확인 할수 있습니다.

검색 옵션에 따라 해당 옵션에 맞는 검색결과를 확인 할 수 있습니다.

  • 구매SW명 : 구매 SW명을 기준으로 검색이 가능합니다.
  • SW명 : SWeeper DB에 등록된 소프트웨어 명을 기준으로 검색이 가능합니다.
  • 구매처 : 해당 SW 라이선스를 구매한 구매처를 기준으로 검색이 가능합니다.
  • 구매부서 : 해당 SW 라이선스를 구매한 부서를 기준으로 검색이 가능합니다.
  • 구매부서(하위포함) : 헤당 SW 라이선스를 구매한 부서의 하위 부서까지 포함하여 검색이 가능합니다.
  • 제조사 : 해당 SW의 제조사를 기준으로 검색이 가능합니다.

Excel 업로드 방법

참고사항 

사용자에 라이선스 할당을 위해서는, 소프트웨어 라이선스 구매 등록시 SWDB와의 매핑이 필요합니다.

  1. 마우스 우클릭하여 엑셀업로드 메뉴를 선택합니다.

  1. 엑셀 업로드 Wizard가 팝업되며, "양식 다운로드"에서 파일을 내려받아 내용에 맞도록 편집하여 저장합니다.

  1. 엑셀 업로드 버튼을 클릭하여 저장된 Excle 파일을 선택합니다.

  1. 업로드된 내용을 확인 후 다음 버튼을 클릭하면 처리결과가 표시되며 완료 버튼을 클릭하면 작업이 완료됩니다.

참고사항 

  1. 우클릭 파일 내보내기(양식 다운로드)시, 해당 PC에 다운받으려는 엑셀 버전이 미리 설치되어 있어야 합니다.
  2. 다운받은 엑셀파일을 임의로 변경해서는 안되며, 입력 양식의 실수로 인한 실패시에는 결과 항목에 실패원인이 출력됩니다.

  1. 다운받은 엑셀파일에서 진한 회색이 된 곳은, 필수 입력 컬럼입니다.

주의사항 

내려받은 엑셀샘플 양식의 서식이 변경되면, 업로드시 문제가 될 수 있습니다. 

다른 파일에서 Copy & Paste 시, 의도치 않게 서식이 변경되어 오류가 발생됨을 주의하시기 바랍니다.

(대표적인 예)

구매일과 만료일 입력시, 구매일 (텍스트)을 날자로 변경되었을 경우, 실제 데이터 오류의 예

  • 셀서식을 임의로 날짜로 변경

  • 업로드시 데이터 오류
