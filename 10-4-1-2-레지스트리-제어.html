10-4-1-2. 레지스트리 제어

10. 기타 ›› 10-4. 명령전송 ›› 10-4-1. 일반명령 ››

장비의 특정 레지스트리 정보를 추가/변경/삭제 할 수 있는 명령입니다. 레지스트리 변경은 장비 장애를 유발시킬 수 있으므로 충분히 확인 후 사용하시기 바랍니다.

설정방법

최초 명령 메뉴 선택에 대한 부분은 상위 목차인 [명령전송]을 확인하여 주시기 바랍니다. 

  1. 레지스트리 제어를 선택합니다.  

  2. 레지스트리 제어에 필요한 설정을 합니다.

  3. 제어구분

  4. 검색 : 정보 존재여부만 확인합니다. 

  5. 명령전송결과 "완료" : 해당 레지스트리 정보가 존재합니다.

  6. 명령전송결과 "실패" : 해당 레지스트리 정보가 존재하지 않습니다.

  7. 추가/수정 : 레지스트리 내용을 추가하거나 변경할 수 있습니다. 

  8. 추가/수정은 정보를 변경할 수 있도록 데이터형식과 값데이터를 설정할 수 있습니다.

  9. 삭제 : 레지스트리 정보를 삭제할 수 있습니다. 

  10. ROOT : 레지스트리 항목 중 ROOT에 해당하는 목록을 표시하며 변경이 필요한 목록을 선택합니다. 

  11. 키 : 레지스트리 키를 입력합니다.
  12. 값 이름 : 키에 해당하는 값을 입력합니다.
  13. 적용 대상 : Windows 32 Bit 또는 64 Bit 중 대상에 맞도록 선택합니다.
  14. 데이터 형식 : 값 이름의 데이터 형식을 선택합니다.

  15. 값 데이터 : 값 이름의 데이터 정보를 입력합니다.

  16. 설정이 완료되면 다음버튼을 클릭합니다.

  17. 이후 설정내용은 [명령전송]을 확인하여 주시기 바랍니다.

주의사항

Windows 레지스트리 정보는 잘못 변경할 경우 시스템 장애를 발생시킬 수 있으므로 충분히 내용 확인 후 사용하시기 바랍니다. 

참고사항

Windows에서 레지스트 편집기를 이용한 편집 방법을 참고합니다. 

레지스트리 편집기는 Windows 명령프롬프트 또는 실행에서 "regedit"를 입력하시면 레지스트리 편집기가 실행됩니다. 

SWeeper의 레지스트리 제어는 손쉽게 원격지 PC의 레지스트리 정보를 검색하고 추가/수정/삭제 할 수 있습니다. 

  1. 키 생성

  2. 값 생성

  3. 값 데이터 입력

참고사항

Mac Agent 는 해당 명령을 지원하지 않습니다.
