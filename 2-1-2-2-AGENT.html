사용자 PC에 설치된 에이전트가 기본적인 수행을 위해 필요한 여러가지 설정을 할 수 있습니다. 

Agent 포트 : 에이전트에서 서버와 통신하기 위한 대기포트를 설정합니다.

보고 주기/명령 요청 주기(분) : 에이전트가 서버로 보고하는 주기와 또는 에이전트가 수행할 명령을 요청하는 주기를 설정합니다. 

  • 보고 시간은 향후 PC 사용시간을 산정하는 Data로 사용됩니다.
  • 최대 설정시간 : 60분

AGENT 버전 : 에이전트 현재 버전을 표시합니다.

  • 처음 서버를 설치한 경우에는 표시되지 않습니다.
  • 담당 엔지니어를 통해 에이전트 패치가 있었을 경우에만 표시됩니다.

AGENT 버전 (MAC) : MAC 에이전트 현재 버전을 표시합니다.

  • 처음 서버를 설치한 경우에는 표시되지 않습니다.
  • 담당 엔지니어를 통해 에이전트 패치가 있었을 경우에만 표시됩니다.

초기 부서 : 에이전트 설치 후 기본 설정될 부서를 설정합니다. 

장기미사용장비전환기준(일) : 장기 미사용장비로 전환되는 기준일을 설정합니다.

  • 보고시간을 기준으로 하며 최근 보고일자 기준으로 설정된 전환 기준일만큼 보고가 없을 경우 장기미사용 상태로 변경됩니다.
  • 최대 설정시간 : 180분

설치유도 여부 : 설치유도 여부를 설정합니다. 

수신메시지 삭제 (일) : 설정한 일수가 지난 메시지를 삭제합니다. 

  • 0 (기본) : 메시지를 삭제하지 않습니다.

  • 0으로 설정시 사용자 화면에서  삭제일 항목이 활성화 되어, 사용자 지정 가능합니다. (12-3-2-3. 환경설정 참고)

  • n일 : 현재일 기준 n+1 일 이전 메시지를 삭제합니다.

  • 사용자 화면에서는 수신메시지 삭제일 항목이 비활성화 됩니다. (관리자 설정에 따라 동작)

  • 최대 설정시간 : 180일

네트워크검색설정

  • [일반정보 - Agent - 네트워크검색 설정]에서 Agent는 네트워크 정보를 검색하여 서버에 보고하는데, 이에 대한 설정입니다.

  • 검색 프로토콜 : 검색에 포함할 프로토콜을 선택합니다.

  • SNMP : SNMP 설정이 되어 있는 Network, 프린터 장비검색을 위해 사용됩니다. (SNMP 인증인 public으로 합니다.)

  • NetBIOS : 장비의 컴퓨터명, 워크그룹을 검색하기 위해 사용합니다.

  • 검색 쓰레드개수 : 검색에 사용할 쓰레드 개수를 설정합니다.

  • 쓰레드 개수가 많을 수록 검색 장비 PC의 성능저하가 발생할 수 있으므로 5 이상 설정하지 않을 것을 권장합니다.

  • 검색 주기 (분) : 네트워크 검색 시간 주기를 설정합니다.

  • 검색 시간이 짧을 수록 실시간 검색 결과 신뢰성이 높아집니다.

  • 검색네트워크 범위 : 검색할 네트워크 범위를 Network 단위가 아닌 Range로 설정합니다.

네트워크차단 예외

  • 네트워크 범위 : 네트워크 차단시에 연결 허용할 범위를 지정합니다.
