데이터 서버는 명령서버들의 DB 연결을 담당합니다. 하나이상의 서버운영이 가능하며 데이터 서버 메뉴는 이러한 서버들을 관리하기 위해 구현되었습니다.

각 데이터 서버들의 접속과 관련한 보고일자 및 상태를 통해 운영 상태를 확인할 수 있습니다. 

설정방법

  1. 데이터 서버 Tab을 선택한 뒤 추가버튼을 클릭합니다. 서버 정보 등록/수정 화면이 팝업되면 해당 정보를 입력합니다. 

  2. 마스터 서버여부 : 마스터는 메인이 되는 서버 기능을하며 한대만 설정 가능합니다.

  3. 입력이 완료되면 확인 버튼을 클릭합니다.

  4. 추가된 정보를 화면에서 확인합니다.

참고사항 

서버관리를 위해서는 해당 서버에 SWeeper 데이터 서버 프로그램이 설치되어 있어야 합니다.
