2. 홈 ›› 2-1. 환경설정 ›› 2-1-3. 자산 ››

자산기능 사용시, 유용한 설정들을 변경할 수 있습니다. 관리자는 사용자PC에 11-3-1-1. 기본정보의 "사용자 정보입력"과 같은 창을 보낼수 있습니다. 

SWeeper에서는 "사용자 정보입력" 창의 자산 입력이 초기에는 비활성화 되도록 기본설정이 되어있어, 사용자는 입력을 할 수 없도록 하였습니다. 

본 설정에서는 이를 활성화하여 입력하도록 하며, 때에 따라서는 강제로 입력하도록 설정할 수 있습니다. 

  1. 자산 입력 활성화 여부

  2. 자산을 입력할 수 있도록 활성화 합니다.

  3. 자산 입력 필수 여부

  4. 사용자가 자산정보 입력을 필수화 합니다.
