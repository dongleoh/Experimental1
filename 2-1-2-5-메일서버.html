사용자가 관리자에게 SW신청을 했거나, 새로운 장비가 추가되는 등의 이벤트가 발생했을 때, 담당자에게 해당 내용을 알려주기 위해 설정합니다. 

메일서버

  1. 메일서버SSL 사용여부 : SSL(Secure Socket Layer) 사용여부를 설정합니다.
  2. 메일서버계정 : 메일 전송에 사용될 메일 서버 계정을 입력합니다.
  3. 메일서버주소/아이피 : 메일 서버의 도메인 주소 또는 IP Address를 입력합니다.
  4. 메일서버비번: 메일 서버 계정의 비밀번호를 입력합니다.
  5. 메일서버포트 : 메일 서버 대기 포트를 입력합니다.
  6. 이벤트로그 발생 시 메일 전송 여부 : 이벤트 로그 발생 시 메일 알림을 위한 메일 전송 여부를 설정합니다.

알람설정

  1. 로그 데이터 파일 저장 : 콘솔을 실행한 후 발생하는 오류나 알람 정보등을 로컬에 저장할지 여부를 설정합니다.
  2. 화면에 표시할 로그 메시지 수 : 콘솔 작업 메시지 창에 표시되는 로그의 최대 열 수를 설정합니다.

  3. DB 검색 주기 : 주기적으로 알람 정보를 검색하는 주기를 설정합니다.

  4. 최대설정시간 : 1440분

  5. 이벤트 메일 송신 언어 : 발송 메일 언어를 설정합니다. (현재는 한국어만 설정가능합니다.)

  6. 이벤트 메일 수신 설정 : 특정 이벤트가 발생할 때 발송되는 메일 정보를 설정합니다.

  7. 2-1-2-5-1 에서 자세히 설명합니다.

하드디스크

  1. 디스크 임계치 (%) : 에이전트 PC가 입력한 임계치를 초과할 경우 관리자 알림 및 메일이 전송됩니다.

  2. 최대 임계치 : 100%

참고사항 

  • Windows : NONE, TLS 지원
  • Linux : TLS, SSL 지원

  • Windows, Linux 서버에서 TLS 로 설정시, 참고사항입니다.

  • SSL 선택

  • Gmail.com 포트 : 587

  • Gmail 이용시, 구글계정관리 - 보안 - "보안 수준이 낮은 앱의 엑세스"  사용 체크

  1. 데이터서버 재시작 또는 즉시반영
