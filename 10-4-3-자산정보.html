10-4-3. 자산정보

10. 기타 ›› 10-4. 명령전송 ››

자산정보(HW 장치, 소프트웨어 설치, SDC 정보 등) 수집을 위한 명령입니다. 

설정방법

단순한 정보 수집명령이므로 특별한 설정 없이 명령 전송하면 됩니다. 

  1. 자산정보 메뉴 중 필요한 정보 수집이 필요한 명령을 선택합니다. 

  2. 하드웨어 검색 : PC의 하드웨어 장치정보를 검색합니다.

  3. 에이전트 최초 설치 시 HW 작동기능이 설정되었다면 자동으로 검색됩니다.

  4. SW 검색 : SWDB를 기준으로 한 설치 정보를 검색하여 결과를 서버에 전송합니다.

  5. 제어판 검색 : 제어판의 프로그램 설치 정보를 검색하여 결과를 서버에 전송합니다.
  6. +DB 기반 검색 : +DB를 기준으로 한 설치 정보를 검색하여, 결과를 서버에 전송합니다.
  7. 라이선스 적용 : 할당하거나 설정된 라이선스 정보를 적용합니다.

  8. 폰트 검색 : 폰트 설치 정보를 검색하여 결과를 서버에 전송합니다.

  9. 명령 선택 후 다음버튼을 클릭합니다.

  10. 이후 설정내용은 [명령전송]을 확인하여 주시기 바랍니다.
