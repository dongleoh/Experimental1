11-3-1-4. SW 자료실

11. CLIENT UI(AGENT) ›› 11-3. 사용자 UI ›› 11-3-1. 장비정보 ››

SW자료실은 사용자가 필요한 SW를 다운받을 수 있도록, 관리자가 등록한 SW입니다. 

(6-1-1. 패키지생성) 관리자의 패키지 설정에 따라, 사용자 UI에 노출되기도 하고, 노출된 SW를 다양한 Case별로 다운받고 설치할 수 있도록 하는 등의 제어가 가능합니다.

(위 : Windows Agent, 아래 : Mac Agent) 

SW 설치방법 

  1. 설치 버튼을 누릅니다.

  

  1. 다운로드 설치 창에서 다운로드를 누릅니다.

  

  1. 다운로드 후 설치가 진행 됩니다.(관리자가 프로그램 실행되도록 설정한 경우)

  

참고사항 

  • 관리자가 패키지를 등록한 방법에 따라, 위의 경우와 같이 다운로드와 동시에 실행 여부가 결정됩니다.
  • 다운만 받도록 설정된 경우, 프로그램 다운 완료시 해당 디렉터리가 열립니다.
  • 라이선스와 연계된 경우, 라이선스 없는 사용자가 설치를 시도하는 경우 아래와 같이 팝업되어 프로그램 다운받을 수 없습니다.

          

  • 실행조건에 일치되지 않는 경우, 역시 프로그램을 다운받을 수 없습니다.

  

  • 이미 설치된 경우 프로그램을 다운받을 수 없습니다.
