(위 : Windows Agent, 아래 : Mac Agent) 

아래의 진단 이미지를 누르면 나타나는 화면입니다.

  1. 우측 상단의 "진단시작" 버튼을 클릭합니다.

  1. 진단이 시작되면, 6단계로 진단하며 그 결과를 보여줍니다. (Mac 은 5단계)

  2. 진단의 종류

  3. Agent 서비스 등록상태 진단

  4. Agent 프로세스 구동상태 진단
  5. 정책수행 프로세스 점검 구동상태 진단
