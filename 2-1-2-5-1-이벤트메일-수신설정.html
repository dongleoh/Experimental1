2. 홈 ›› 2-1. 환경설정 ›› 2-1-2. 일반설정 ›› 2-1-2-5. 메일서버 ››

특정 이벤트가 발생했을 때, 관리자에게 각 이벤트별 어떠한 내용으로 알림메일을 발송할 지에 대한 설정을 합니다. 이렇게 설정된 이벤트들은 지정된 관리자에게 자동으로 메일 발송이 됩니다.  메시지를 받기 위한 관리자 설정은 9-1-1. 관리자 관리에서 설명합니다.

이벤트 메일 수신설정

  1. 해당 항목을 클릭합니다.

  1. 각 항목에 대해 수정

  • 설명

  • [@Event]

- EquipNew(신규장비), SwRequest(SW신청), AssetNew(신규자산) 같은 이름으로 표기

  • [@EquipId]

- 장비키가 표기

  • [@Time]

- 발생시간 표기

  • [@Contents]

- Contents 내용 표기

  • [@IP]

- IP 표기

  • [@User]

-사용자 표기

  • [@Dept]

-부서 표기

  • [@LicenseType]

- 라이선스타입 표기

  1. 저장

참고사항 

  1. 이벤트 메일을 수신하기 위해서는 메일서버 설정이 반드시 필요합니다. 

  2. 현재 이벤트는 아래 5개에 대해서만 가능합니다.

- 신규자산

  • 새로운 자산이 추가된 경우

- 장비신규 설치

  • 새로운 장비에 에이전트를 설치한 경우

- 디스크 임계치 초과

  • 디스크 임계치가 초과한 경우

- SW신청

  • 사용자가 SW를 신청한 경우

- 소프트웨어 구매 만료

  • 소프트웨어 만료일이 15일 이하인 경우

  • 이벤트 메일 수신 예
