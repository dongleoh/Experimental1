5-1-1-2. 소프트웨어 리소스 생성 확인

5. 소프트웨어 ›› 5-1. 자산관리 ›› 5-1-1. 라이선스 ››

아래와 같이 기업에서 구매한 소프트웨어 정보를 입력하고, 매핑(소프트웨어명)하면, 구매수량에 입력한 수량만큼 하단창에 리소스가 생성됩니다.

  1. 소프트웨어 구매 등록

  1. 생성된 리소스ID

(HelpNDoc 구매수량 10개 만큼의 리소스ID 10개 생성된 화면)

소프트웨어 설정 컬럼 설명

  • 만료                : 라이선스 만료 여부
  • 리소스 ID        : 할당할수 있는 라이선스
  • 할당부서        : 해당 리소스를 관리하는 담당부서
  • 자동할당        : 해당 소프트웨어가 할당된 부서의 장비에 설치시 자동으로 라이선스를 할당
  • 하위부서포함        : 하위부서의 장비(사용자)에도 라이선스 할당할지 여부
  • 인가일        : 장비(사용자)에 라이선스 할당한 날짜
  • 시리얼키        : 소프트웨어 라이선스 구매시, 받은 라이선스 키 입력
  • 설치                : 라이선스 할당한 장비(사용자)가 해당 소프트웨어 설치 여부
  • 할당방식        : 라이선스 할당 방식(장비/사용자)
  • 인가대상부서        : 라이선스 할당한 부서
