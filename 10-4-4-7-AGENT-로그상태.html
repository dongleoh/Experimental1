10-4-4-7. AGENT 로그상태

10. 기타 ›› 10-4. 명령전송 ›› 10-4-4. 장비 설정 변경 ››

장비의 로그 설정 상태를 확인할  수 있는 명령입니다. 

설정방법

최초 명령 메뉴 선택에 대한 부분은 상위 목차인 [명령전송]을 확인하여 주시기 바랍니다. 

  1. Agent 로그 상태를 선택합니다.

  1. 별도 설정이 없으므로 다음 버튼을 클릭합니다.
  2. 명령 처리 상태가 완료가 되면 [명령결과] 탭에서 장비를 선택하여 명령 설정 상태를 확인합니다.

참고사항

Mac Agent 는 해당 명령을 지원하지 않습니다.
