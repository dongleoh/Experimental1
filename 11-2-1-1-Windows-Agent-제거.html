에이전트 제거의 방법으로 사용자가 AGENT 제거를 신청하고 관리자가 승인하는 방법과 관리자가 명령을 통해 제거하는 방법  2가지를 제공합니다. 

<사용자 요청에 의한 제거>

시작메뉴에서 "SWeeper Agent 제거"을 실행하거나, 에이전트 설치경로의 "nxrmagt.exe" 파일을 실행합니다.

  • 삭제 요청

  • 삭제키 우측의 요청버튼을 누릅니다.

  • 에이전트 삭제요청은 관리자에 의해 승인/거절될 수 있습니다.
  • 관리자가 정상적으로 승인하면, 사용자가 요청버튼을 다시 눌러 삭제키를 자동으로 가져올 수 있습니다.
  • 승인되지 않은 요청에 대해서는 에이전트 삭제가 불가능합니다.

  • 삭제

  • 삭제키를 정상적으로 입력한 후에는 하단의 삭제 버튼을 눌러 에이전트를 삭제합니다.

  • 시스템 재부팅

  • 정삭적인 프로그램 제거를 위해 시스템을 재부팅하는 걸 권장합니다.

<관리자  명령에 의한 제거>

메뉴얼 10-4-4-4 를 참고합니다.

참고사항 

<임시키>

  • 서버와 통신이 불가능한 상태에서 에이전트를 설치한 경우 임시키를 통해 삭제합니다.

  • 서버와 이미 통신이 된 장비에는 임시키가 나오지 않습니다.

  • 서버와의 통신 시도하는 과정이 있어, 임시키 생성까지는 약 5분정도 소요됩니다.

<마스터삭제키>

  • 긴급한 경우 사용합니다. 담당 엔지니어에게 문의바랍니다.
