실행현황

  • 불법SW : 라이선스를 할당받지 않은 소프트웨어의 실행정보
  • 합법SW : 라이선스를 할당받은 소프트웨어의 실행정보
  • 임시SW : 임시라이선스를 할당받은 소프트웨어의 실행정보
  • 미분류 : 과거에 설치 실행되었지만 현재 설치 확인되지않은 실행정보

 

조회 방법 

  1. 불법 SW 또는 휴면 라이선스를 선택합니다.
  2. 우측 화면에 관련 소프트웨어 리스트가 조회됩니다.
  3. 실행현황을 보고자하는 소프트웨어를 선택합니다.

  4. 화면 하단에 관련 결과가 조회됩니다.

  5. 소프트웨어 사용장비 목록 : 장비와 사용시간이 조회됩니다.

  6. 실행 프로세스 목록 : 장비가 실행한 사용 프로세스와 사용시간이 조회됩니다.
