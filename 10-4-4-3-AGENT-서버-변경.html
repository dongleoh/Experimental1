10-4-4-3. AGENT 서버 변경

10. 기타 ›› 10-4. 명령전송 ›› 10-4-4. 장비 설정 변경 ››

서버가 추가되어 에이전트 분산을 시키거나 서버 IP Address가 변경된 경우 에이전트의 접속 서버정보를 변경하기 위한 명령입니다. 

🛈 주의사항

  • 서버 변경은 환경설정과 명령전달에 있어서 주의를 요하므로 반드시 담당 엔지니어와 함께 협의 후 진행하시기 바랍니다.

설정방법

최초 명령 메뉴 선택에 대한 부분은 상위 목차인 [명령전송]을 확인하여 주시기 바랍니다. 

  1. AGENT 서버 변경을 선택합니다. 

  2. 추가된 서버 또는 변경된 서버를 선택합니다.

  3. 설정이 완료되면 다음 버튼을 클릭합니다.
  4. 이후 설정내용은 [명령전송]을 확인하여 주시기 바랍니다.
