설치

  • 설치 언어 선택

  • 설치 언어를 선택하고 다음 버튼을 누릅니다.

  • SWeeper Agent

  • 명령서버의 IP정보를 입력하고 다음 버튼을 누릅니다.

  • 설치할 위치 선택

  • Agent를 설치할 경로를 입력하고 다음 버튼을 누릅니다.

  • 시작 메뉴 폴더 선택

  • 시작메뉴에 생성할 폴더를 입력(선택)하고 다음 버튼을 누릅니다.

  • 추가 사항 적용

  • 빠른실행 아이콘, 제거아이콘 생성 여부를 체크하고 다음 버튼을 누릅니다.

  • 설정 미체크시에는 시작메뉴에 아이콘이 생성되지 않습니다.
