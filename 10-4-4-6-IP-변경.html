10-4-4-6. IP 변경

10. 기타 ›› 10-4. 명령전송 ›› 10-4-4. 장비 설정 변경 ››

장비의 IP Address 정보를 변경할 수 있는 명령입니다. 

주의사항 

  • 서브넷과 게이트웨어 정보를 변경할 수 없으므로  동일 Network Class 내에서만 변경해야 합니다.
  • 복수 설정이 불가능하며 한번에 한대의 장비만 변경 가능합니다.
  • 변경 명령을 전송했으나 서버와 통신이 불가능할 경우 이전 IP정보로 복원됩니다.

설정방법

최초 명령 메뉴 선택에 대한 부분은 상위 목차인 [명령전송]을 확인하여 주시기 바랍니다. 

  1. IP 변경을 선택합니다. 

  2. 기존 IP와 바꿀 IP 정보를 입력합니다.

  3. 설정이 완료되면 다음 버튼을 클릭합니다.
  4. 이후 설정내용은 [명령전송]을 확인하여 주시기 바랍니다.
