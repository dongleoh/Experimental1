9-1-3-2. 사용자 현황

사용자에 대한 전체적인 현황을 확인할 수 있습니다.

  • 설치현황

  • 현황

  • 설치 : 사용자의 PC에 설치한 모든 소프트웨어의 종류를 확인합니다.

  • 불법 : 사용자의 PC에 불법으로 설치한 모든 소프트웨어의 종류를 확인합니다.
  • 합법 : 사용자의 PC에 합법적인 라이선스를 가진 소프트웨어의 종류를 확인합니다.

  • 임시 : 사용자의 PC에 임시 라이선스를 받은 소프트웨어의 종류를 확인합니다.

  • 라이선스

  • 할당 : 구매한 소프트웨어를 장비/사용자에 할당한 사항을 확인합니다.

  • 선할당 : 장비에 해당 소프트웨어를 설치하지 않았지만, 라이선스를 미리 할당한 사항을 확인합니다.

  • 불법원인

  • 비구매 : 구매하지 않았음을 의미합니다.

  • 구매초과 : 구매한 소프트웨어보다 더 많이 설치되었음을 의미합니다.
  • 미인가 : 할당할수 있는 라이선스를 보유한 상태지만, 관리자의 승인없이 불법 설치하였음을 의미합니다.

  • 비인가 : 회사에서는 사용할 수 없는 소프트웨어로 분류되어졌음을 의미합니다.

  • 사용분석

  • 실행

  • 합법사용 : 합법적으로 설치된 소프트웨어의 실행시간이 표시됩니다.

  • 라이선스를 할당했지만, 효율적으로 사용하지 않는 사용자를 확인할 수 있습니다.

  • 불법사용 : 불법적으로 설치된 소프트웨어의 실행시간이 표시됩니다.

  • 불법 사용자의 이용실태를 확인할 수 있습니다.

  • 휴면라이선스

  • 합법 : 합법으로 설치되었다가 휴면관리 대상에 지정된 소프트웨어의 실행시간이 표시됩니다.

  • 불법 : 불법으로 설치되었다가 휴면관리 대상에 지정된 소프트웨어의 실행시간이 표시됩니다.

  • 통제현황

  • 차단 : 프로그램 실행중 차단된 건수가 표시됩니다.

  • 설치 : 설치된 프로그램의 건수가 표시됩니다.

  • 실행 : 샐행된 프로그램의 건수가 표시됩니다.

  • 구매만료예정

  • 구매 만료일(30일전)이 다가오는 소프트웨어의 건수가 표시됩니다.

  • 삭제이력

  • 소프트웨어가 삭제된 건수가 표시됩니다.

참고사항 

  • 표시된 모든 수치를 선택(클릭)하면 해당 내용을 자세히 확인할 수 있습니다.
  • 통제현황(차단/설치/실행), 삭제이력, 사용분석(실행)의 조회기간은 기본 일주일이며 관리자가 설정할 수도 있습니다. (2-1-4. 소프트웨어)
