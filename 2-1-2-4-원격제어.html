2-1-2-4. 원격제어

2. 홈 ›› 2-1. 환경설정 ›› 2-1-2. 일반설정 ››

원격제어에 대한 환경설정을 할 수 있습니다.

  1. 원격지원 표시 여부 : PC에 원격지원을 했을 경우 화면에 "원격지원 중"이라는 상태 표시 여부를 설정합니다.
  2. 원격지원 종료 메시지 표시 여부 : 원격지원 완료 후 종료 시 원격 접속 시간 및 관리자 정보를 표시하는 메시지 팝업 여부를 설정합니다.
  3. 원격제어 시 차단 소프트웨어 목록 : 원격제어 시 보안을 위한 특정 실행파일을 실행할 수 없도록 하는 설정을 합니다.

참고사항 

  1. 강제 원격제어시에는 "원격지원 표시 여부"와 "원격지원 종료 메시지 표시여부" 옵션에 관계없이, 무조건 표시됩니다.

  2. 원격제어시, 콘솔 관리자 PC에 TCP 8230~8300 중 미사용 포트가 Listening 되며 에이전트 접속 대기합니다. 

원격제어 사용 포트가 차단되지 않도록 방화벽 설정을 점검 바랍니다.

  1. 다른 사람이 해당 PC에 원격데스크톱으로 연결한 경우, 원격제어 동의 메시지창이 사용자에게 전달되지 않아, 원격제어가 불가능합니다.

  2. 관리콘솔 로그인 접속 아이피와 동일한 접속 IP 주소를 가진 장비는 원격 제어할 수 없습니다.

원격제어 참고사항 

  • 원격제어화면에서 "Function-리부팅 후 재접속" 옵션은 에이전트 장비가 30초(5초 간격 6회) 이내에 게스트(관리자)와 연결되지 않으면 관리자 화면에서 무한대기하게 되는 경우가 있습니다. 

재기동시 네트워크 카드가 늦게 활성화되는 경우에도 무한대기 현상이 생길수 있습니다.  이때는 다시 연결을 시도하시기 바랍니다.
