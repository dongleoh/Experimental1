장비의 로그  설정을 할  수 있는 명령입니다. 

에이전트 모니터링이 필요할 때, 로그 레벨/유형/경로/파일사이즈/갯수 옵션을 설정하여 원하는대로 로그를 확인할 수 있습니다. 

설정방법

최초 명령 메뉴 선택에 대한 부분은 상위 목차인 [명령전송]을 확인하여 주시기 바랍니다. 

  1. Agent 로그 설정을 선택합니다.

  1. 로그 설정에 필요한 설정을 합니다.

  2. 작동기능 : 작동기능 별 로그 설정을 할 수 있습니다.

  3. 로그레벨 :  로그 수준을 설정합니다.

  4. 로그유형 : 파일 사이즈별 / 일자별 중 선택합니다.

  5. 파일 사이즈별 : '로그파일 사이즈'에 입력한 용량 이상이 되면 이전 내용은 저장하고 새 파일에 기록합니다.

  6. 일자별 : 하루 단위로 로그를 기록합니다. 파일 크기는 제한하지 않고 날짜가 변경되면 이전 내용은 저장하고 새 파일에 기록합니다.

  7. 로그파일 사이즈 : 로그파일 크기가 무제한 커지는 것을 방지하기 위해 최대 크기를 지정합니다.

  8. 로그파일 백업갯수 : 로그 파일 기록 파일 개수를 설정합니다.
  9. 로그파일 경로 : 로그 파일을 저장할 폴더를 설정합니다.

  10. 로그 초기화 일자 : 설정한 로그 세팅은 설정한 날짜부터 ~ 초기화일자 전날까지 적용되며 '로그 초기화 날짜' 이후에는 로그 기본설정으로 되돌아갑니다.

  11. 설정이 완료되면 다음버튼을 클릭합니다.

  12. 이후 설정내용은 [명령전송]을 확인하여 주시기 바랍니다.

참고사항

예시

  • 작동기능 : 전체
  • 로그 레벨 : INFO
  • 로그 유형 : 파일 사이즈별
  • 로그파일 사이즈 : 2,000,000 Byte
  • 로그파일 백업 갯수 : 5
  • 로그파일 경로 : D:\Test
  • 로그 초기화 일자 : 2021-12-13

예시설명

  • 전체 작동기능 로그가 D:\Test경로에 기록하며 각 기능로 로그 파일 크기가 2MB가 넘으면 새 파일에 기록합니다. 이전 로그는 총 5개의 파일까지 기록합니다.
  • 이 설정은 2021-12-13일이 되면 모두 초기화 됩니다. 기본 로그 설정으로 재설정됩니다.
  • 참고이미지

Mac Agent 는 해당 명령을 지원하지 않습니다.
