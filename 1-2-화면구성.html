1-2. 화면구성

1. 개요 ››

관리콘솔에 로그인하였을 때, 기본화면에 대한 설명입니다.

  • 관리콘솔 설치하면, 로그인 화면이 팝업됩니다.

  • 로그인 정보 입력 내용

  • 아    이     디 : 생성한 계정을 입력합니다.

  • 패  스  워  드 : 패스워드를 입력합니다.
  • 서  버  선  택 : SWeeper DataServer IP를 입력합니다. 
                     한번 입력된 서버IP는 저장되어, 재 로그인 시 기본입력이 됩니다.
  • 포  트  선  택 : 포트번호  "8220"을 입력합니다.

  • 접속   아이피 : PC에 설정된 IP를 입력합니다. 
                        자동으로 표시되지만 선택이 잘못된 경우 원격제어 기능을 사용할 수 없으므로 사용중인 IP를 확인하여 선택하시기 바랍니다.

  • 콘솔화면

SWeeper 콘솔 화면 구성은 아래 그림과 같이 6가지로 분류되어 있습니다. 

  • 화면구성

  • 리본메뉴 : 직관적인 아이콘으로 각 메뉴를 표시를 하여 관리자가 쉽게 선택하여 기능을 사용할 수 있도록 했습니다.

  • 조직도 : 부서, 네트워크 등의 분류를 표시하는 화면으로 장비 설치현황, 소프트웨어 현황 등과 연계하여 조회되도록 합니다.
  • 메인화면 : 각 기능별 조회 결과가 표시되는 화면으로 리스트 또는 수량 등 다양한 정보를 표시하게 됩니다.
  • 속성창 : 메인화면에서 조회되는 각 정보에 대한 상세한 내용을 추가적으로 표시합니다.

  • 상태표시줄 : 관리콘솔의 현재 작업 상태를 표시합니다.

  • Server IP : 현재 접속한 SWeeper 서버의 IP 정보를 표시합니다.

  • 로그인아이디 : 콘솔 로그인한 계정을 표시합니다.
  • 상태 메시지 : 관리 콘솔에서 발생하는 작업 메시지를 출력합니다.

  • 예약 로그오프 : 관리자 접속 보안을 위한 로그 오프 예약 시간을 표시합니다.

  • 60분이 지나면, 프로그램 종료 메시지가 제공됩니다. 이때 확인을 누르면 자동 60분이 연장되고, 그렇지 않으면 프로그램 종료됩니다.

  • 접속시간 : 로그인한 시간을 표시합니다.

  • 한국어 : 현재 관리콘솔에 표시되는 언어 정보를 표시합니다.

  • 조직도 메뉴 : 단축 아이콘(보이기/감추기)

참고사항 

  • 톱니바퀴 아이콘을 통해 상태 표시줄에 표시할 항목을 설정할 수 있습니다.

  • 컬럼영역에서 마우스 우클릭 시 "열선택"에서  메인화면에서 노출될 열을 선택 할 수 있습니다.

s

  • 열 설정 기능을 이용하여 그리드뷰 열설정을 편하게 할 수 있습니다.
